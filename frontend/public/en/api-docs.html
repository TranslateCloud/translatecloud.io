API Overview

The TranslateCloud API is organized around REST. Our API has predictable resource-oriented URLs, accepts JSON-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

Base URL: https://api.translatecloud.io/v1
API Version: v1
Format: JSON

Authentication

Authenticate your API requests by including your API key in the Authorization header. You can manage your API keys in your dashboard.

cURL 
curl https://api.translatecloud.io/v1/translate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, world!",
    "target_language": "es"
  }'
Keep your API key secure

Do not share your API key in publicly accessible areas such as GitHub, client-side code, etc.

Rate Limiting

The API implements rate limiting to ensure fair usage. Rate limits vary by plan:

Plan Requests/minute Characters/month
Free 10 50,000
Starter 60 1,000,000
Professional 300 10,000,000
Enterprise Custom Unlimited

When rate limited, the API returns a 429 Too Many Requests response with:

{
  "error": {
    "code": "rate_limit_exceeded",
    "message": "Rate limit exceeded. Try again in 23 seconds.",
    "retry_after": 23
  }
}

POST /translate

Translate a single text string to a target language.

POST
https://api.translatecloud.io/v1/translate

Request Body

Parameter Type Required Description
text string Yes Text to translate (max 5,000 characters)
target_language string Yes ISO 639-1 language code (e.g., "es", "fr", "de")
source_language string No Source language (auto-detected if not provided)
preserve_html boolean No Preserve HTML tags in translation (default: false)
formality string No "formal" or "informal" (default: neutral)

Example Request

JavaScript 
const response = await fetch('https://api.translatecloud.io/v1/translate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Welcome to our platform',
    source_language: 'en',
    target_language: 'es',
    formality: 'formal'
  })
});

const data = await response.json();
console.log(data);

Response

{
  "translated_text": "Bienvenido a nuestra plataforma",
  "source_language": "en",
  "target_language": "es",
  "characters_used": 26,
  "detected_language": "en",
  "confidence": 0.98
}

POST /translate/batch

Translate multiple text strings in a single request for improved efficiency.

POST
https://api.translatecloud.io/v1/translate/batch

Request Body

Parameter Type Required Description
texts array Yes Array of strings to translate (max 100 items)
target_language string Yes ISO 639-1 language code
source_language string No Source language (auto-detected if not provided)

Example Request

Python 
import requests

response = requests.post(
    'https://api.translatecloud.io/v1/translate/batch',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'texts': ['Home', 'About Us', 'Contact', 'Pricing'],
        'target_language': 'fr'
    }
)

print(response.json())

Response

{
  "translations": [
    { "original": "Home", "translated": "Accueil" },
    { "original": "About Us", "translated": "À propos de nous" },
    { "original": "Contact", "translated": "Contact" },
    { "original": "Pricing", "translated": "Tarification" }
  ],
  "target_language": "fr",
  "total_characters": 32
}

POST /translate/html

Translate an entire HTML document while preserving structure, tags, and SEO elements.

POST
https://api.translatecloud.io/v1/translate/html

Request Body

Parameter Type Required Description
html string Yes HTML content to translate
target_language string Yes ISO 639-1 language code
preserve_seo boolean No Translate meta tags, alt text, etc. (default: true)
exclude_selectors array No CSS selectors to exclude from translation

Example Request

Node.js 
const axios = require('axios');

const html = `
  <html>
    <head>
      <title>Welcome</title>
      <meta name="description" content="Join our platform">
    </head>
    <body>
      <h1>Hello World</h1>
      <p>This is a test.</p>
    </body>
  </html>
`;

const response = await axios.post(
  'https://api.translatecloud.io/v1/translate/html',
  {
    html: html,
    target_language: 'de',
    preserve_seo: true,
    exclude_selectors: ['.no-translate', '#skip-translation']
  },
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  }
);

console.log(response.data);

Response

{
  "translated_html": "<html>...</html>",
  "target_language": "de",
  "elements_translated": 4,
  "characters_used": 87
}

GET /projects

List all translation projects for your account.

GET
https://api.translatecloud.io/v1/projects

Query Parameters

Parameter Type Description

Webhooks

Configure webhooks to receive real-time notifications about translation events.

Setting Up Webhooks

POST https://api.translatecloud.io/v1/webhooks

{
  "url": "https://your-app.com/webhooks/translations",
  "events": [
    "translation.completed",
    "translation.failed",
    "project.completed"
  ],
  "secret": "your_webhook_secret"
}

Webhook Payload

{
  "id": "evt_xyz789",
  "type": "translation.completed",
  "created": 1642252800,
  "data": {
    "translation_id": "trans_456",
    "project_id": "proj_123",
    "source_language": "en",
    "target_language": "es",
    "characters": 1250,
    "status": "completed"
  }
}

Verifying Webhooks

Verify webhook signatures to ensure they're from TranslateCloud:

const crypto = require('crypto');

function verifyWebhook(payload, signature, secret) {
  const hmac = crypto.createHmac('sha256', secret);
  const digest = hmac.update(payload).digest('hex');
  return crypto.timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(digest)
  );
}

Error Codes

TranslateCloud uses conventional HTTP response codes to indicate the success or failure of an API request.

Code Status Description
200 OK Request succeeded
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
402 Payment Required Insufficient credits or plan limit reached
404 Not Found Resource doesn't exist
429 Too Many Requests Rate limit exceeded
500 Server Error Something went wrong on our end
503 Service Unavailable Temporary outage, retry with backoff

Error Response Format

{
  "error": {
    "code": "invalid_request",
    "message": "The 'target_language' parameter is required.",
    "param": "target_language",
    "type": "validation_error"
  }
}
